


Description of the files

- "IOUseDetail.txt" is the original dataset available on the BEA website as part of the following zipped file:
http://www.bea.gov/industry/zip/2002detail_redef.zip

- "FIOUseDetail.txt" is a description of the content of IOUseDetail.txt (also part of the zipped file above)

- "iousedetail.dta" is the STATA data file corresponding to the first 5 columns of the file IOUseDetail.txt.

- "Do-file to computing Upstreamness.do" is a STATA do-file generating the upstreamness index (from the iousedetail.dta file).
